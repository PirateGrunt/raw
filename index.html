R workshop at the 2014 RPM

This page will host instructions, data and source code for the R workshop at the 2014 RPM seminar. To register, please visit the CAS website.

UPDATE: Several data files have been loaded to the interweb. Please click the button in the bar on the left to download these files. You should have them on your computer for the workshop.

This site will be updated continually leading up to the day of the workshop. If possible, I'll notify everyone of changes by e-mail.

Workshop Schedule

9:00 - 10:30 Section 1 - Getting started
  • What the heck is R?
  • Installation
  • Basic data manipulation
  • Package installation
10:30-10:45 Break
10:45-12:15 Section 2 - Basic analysis
  • Basic visualization
  • Data frames
  • Writing your own functions
  • Flow control
  • More on linear regression
12:15-1:00 Lunch
1:00-2:30 Section 3 - Reserving in R
  • MRMR
  • ChainLadder
2:30-2:45 Break
2:45-3:45 Model development
  • Simulation
  • Fitting a size of loss distribution
  • Logistic regression and GLMs
3:45-4:00 Break
4:00-? Further topics
  • Intermediate visualization
  • Basic data mining
  • Reproducible research

Note that the schedule on the CAS website lists breaks as lasting 30 minutes. Presuming there are no objections, I'd like to try and keep that to 15 minutes. We'll discuss this at the start of the workshop.
